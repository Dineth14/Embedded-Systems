I/O Ports & Communication

GPIO & Analog I/O

GPIO (General Purpose Input/Output) pins can be configured as:

  • Input: floating, pull-up, or pull-down
  • Output: push-pull or open-drain, with configurable speed
  • Alternate Function: UART TX/RX, SPI, I2C, PWM, etc.
  • Analog: ADC input or DAC output

ADC (Analog-to-Digital Converter)

Resolution determines the number of discrete levels:

$$\text{LSB} = \frac{V_{\text{ref}}}{2^n}, \quad \text{Digital Value} = \left\lfloor \frac{V_{\text{in}} \times 2^n}{V_{\text{ref}}} \right\rfloor$$

where $n$ is the number of bits. A 12-bit ADC with $V_{\text{ref}} = 3.3\,\text{V}$ has LSB $= 3.3/4096 \approx 0.8\,\text{mV}$.

UART Protocol

Definition

UART (Universal Asynchronous Receiver/Transmitter) is a serial protocol with no clock line. Both sides agree on a baud rate. Frame format: [Start bit] [Data 5–9 bits] [Parity (optional)] [Stop bit(s)].

$$\text{Bit time} = 1 / \text{Baud rate}, \quad \text{Frame time} = \frac{\text{bits per frame}}{\text{Baud rate}}$$

Common configuration: 115200 8N1 = 115200 baud, 8 data bits, no parity, 1 stop bit. Each character takes 10 bit-times (1 start + 8 data + 1 stop) ≈ 86.8 μs.

SPI Protocol

Definition

SPI (Serial Peripheral Interface) is a synchronous, full-duplex protocol with 4 signals: SCLK (clock), MOSI (Master Out Slave In), MISO (Master In Slave Out), CS (Chip Select, active low). SPI has 4 modes defined by clock polarity (CPOL) and phase (CPHA).

SPI advantages: high speed (up to 100+ MHz), full-duplex, simple hardware. Disadvantages: one CS line per slave, no flow control, no acknowledgement.

I²C Protocol

Definition

I²C (Inter-Integrated Circuit) is a synchronous, half-duplex, multi-master protocol using only 2 wires: SDA (data) and SCL (clock). Each device has a 7-bit address. Supports 100 kHz (standard), 400 kHz (fast), and 3.4 MHz (high-speed) modes.

I2C transaction: [Start] [Address + R/W] [ACK] [Data] [ACK] ... [Stop]. The master always drives SCL; SDA is bidirectional with open-drain drivers and pull-up resistors.
