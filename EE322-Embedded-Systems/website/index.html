Embedded Systems Design

EE 322 — Department of Electrical & Electronic Engineering

University of Peradeniya, Sri Lanka  |  3 Credits  |  27 Lecture Hours

Course Modules

Module 01
Introduction to Embedded Systems
Definitions, application domains, platform comparison, design challenges, and a case study on ABS braking.
2 hrsILO-1
Module 02
Microprocessor & Microcontroller Architecture
Von Neumann vs Harvard, CISC/RISC, pipelining, branch prediction, ARM Cortex-M memory model, and interrupts.
5 hrsILO-1, ILO-2
Module 03
Embedded Processors
MCU/DSP/FPGA/ASIC taxonomy, ARM Cortex families, bare-metal C programming, CMSIS, cross-compilation.
4 hrsILO-2, ILO-3
Module 04
Memory Architectures
SRAM, DRAM, Flash, MRAM, memory hierarchy, cache operation (direct-mapped, N-way), DMA transfers.
2 hrsILO-2
Module 05
I/O Ports & Communication
GPIO, ADC/DAC, PWM, polling vs interrupts, UART, SPI, I²C protocols with timing diagrams.
2 hrsILO-3
Module 06
Multitasking
Super-loop limitations, threads/processes, context switching on Cortex-M, mutexes, semaphores, FreeRTOS.
2 hrsILO-4, ILO-5
Module 07
Task Scheduling
RMS, EDF, DMS algorithms, utilisation bound, response time analysis, Gantt charts, WCET estimation.
2 hrsILO-4, ILO-5
Module 08
Analysis & Verification
Performance profiling, FMEA, reliability metrics, formal verification with SPIN/Promela, MISRA-C guidelines.
4 hrsILO-6, ILO-7
Module 09
Modeling of Embedded Systems
UML diagrams, finite state machines, hierarchical state machines, FSM implementation in C, model-driven dev.
4 hrsILO-7, ILO-8
